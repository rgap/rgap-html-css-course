Table Widths: Percent vs PX

Percent-based columns

20% 50% 30%
Code P100 Scales with container size (percent width). Flexible
Code P200 Resize the window to see columns grow/shrink proportionally. Flexible

Pixel-based columns

250px 450px 300px
Code P100 Fixed pixels do not scale with container width. Fixed
Code P200 If content is wider than the pixel width, browser may expand in auto layout. Fixed
